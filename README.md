# 2025-2_Programacao1
Repositório destinado ao armazenamento e entrega de atividades de Programação I


## Atividade #1
Desenvolver a tela de cadastro de clientes e cadastro de serviços

    Utilizar a tela de Cadastro de Veículos como modelo para desenvolver a atividade.
    Ambas as telas devem conter no mínimo o CRUD, ou seja, ações para Salvar, Alterar, Excluir e Pesquisar.
    No modelo do Cadastro de Veículos, não tem a opção de visulizar os dados no JTable. Então, a atividade também busca desafiá-los a desenvolver essa pesquisa e permitir visualizar os dados do cliente ou serviço informado.
    Opcionalmente, coloque mais atributos que você achar pertinente.

Desenvolver em sala de aula.
Data máxima para entrega:
Valor: 10,0
Forma de avaliação: individual em cada PC e com perguntas técnicas sobre cada ponto desenvolvido.
Não fique com dúvidas. Procure o professor via WhatsApp ou agende uma aula extra no IFG para tirar dúvidas.

## Atividade #3
Desenvolver uma tela de cadastro de produtos com as seguintes especificações

    Atributos: id, nome, descrição, preço, data da última compra e fornecedor.
    A tela deve conter ações para executar o CRUD, ou seja, ações para Salvar, Alterar, Excluir e Pesquisar. Além do componente visual, cada botão deve executar sua respectiva ação no banco de dados.
    Sobre a ação de Pesquisar, a tela deve conter uma opção de visulizar os dados no JTable. Então, a atividade também busca desafiá-los a desenvolver essa pesquisa, permitindo visualizar os dados dos produtos. Além disso, ao dar duplo-clique sobre um produto, deve-se permitir visualizar os dados nas caixas de texto correspondente a cada campo.
    A tela deve ser criada com base nos conceitos de MVC-DAO.
    O projeto deve ser criado a partir de um projeto Maven.

Desenvolver em sala de aula e em casa.
Data máxima para entrega:
Valor: 10,0
Forma de avaliação: individual em cada PC e com perguntas técnicas sobre cada ponto desenvolvido.
Não fique com dúvidas. Procure o professor via WhatsApp ou agende uma aula extra no IFG para tirar dúvidas.
